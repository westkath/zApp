# zStats | Application :sparkles:

zStats is a 2-part project, composed of an application (zApp) and a library (zLibrary) that will make use of the Riot Games API to return information about a selected Summoner.

## Languages + Tools
- [x] Java
- [x] JavaFX (XML, CSS)
- [x] Maven
